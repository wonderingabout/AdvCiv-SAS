This leaderhead is a converted model from Atlantica Online (http://en.wikipedia.org/wiki/Atlantica_Online). Therefore, while slightly modifications of the models and artwork were made by me, I take no credit for both and all rights are with NDOORS Corporation.

This leaderhead uses modified John II animations. It's completely modular. All files in the zip-file are already put into the necessary folders, unzipping it directly in the Mods folder of BtS should result into a working mod. It has a polycount of about 5500 triangles. There are a noshader and a shader version included.

Edit: Thanks to Lemon Merchant there is now also a set of diplomatic text messages included in the download, which can be found in the Grace_OMalley_CIV4GameText.xml file. Since they can't be made modular, they are not activated in this modcomp, but can be added to any mod this modcomp is part of.
